# Python Guide Sheet

This project is a compact reference for the fundamental concepts of Python.

Although I have been practising Python for a long time, I created this guide as
a reminder of the basics: the building blocks that are easy to overlook and
always worth revisiting. It is intentionally simple and practical rather than
an exhaustive Python course.

The material is also useful for teaching. Each example is small enough to read,
run, modify, and discuss, making it suitable for anyone who is starting to
learn Python or wants a clear refresher.

## Contents

The first section currently covers:

- Printing values and using `print()` options
- Basic data types
- Type casting
- Variables and f-strings
- User input
- Simple exercises for practice

More sections can be added over time as a reference for other core Python
concepts.

## How to use this guide

1. Open the files in `01_basic` in numerical order.
2. Run each example and observe the output.
3. Change the values or expressions to see what happens.
4. Complete the exercises without looking at the examples first.

To run a file:

```bash
python guidesheet-python/01_basic/01_print.py
```

## Purpose

This repository is both a personal cheat sheet and a teaching aid: a small,
accessible collection of examples that helps explain Python's essential ideas
clearly.
